About the Ticket System

What was it made for?

The Ticket System I have built was made for a school so that we can gather and structure requests from students, teachers and administrative staff in one central place. The system makes it possible to create, track and handle problems efficiently. Whether it concerns IT Support, Software Development, Network Administration or Account Management, there will always be someone available to solve the problem.

What is the purpose?

The purpose of the system is to create an overview, ensure faster response times and improve communication between users and support functions. By creating all tickets, tasks can be prioritised, responsibilities distributed and solutions documented so that problems can be handled more efficiently in the future.
